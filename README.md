# hoover

A small hoover game app. Built on react-create-app, with additional configuration for css-modules.

Tested: in Chrome, Firefox, Safari, and Edge.

## Initial install

Using  <https://yarnpkg.com/>
```
  yarn
```  

## Development
```
  yarn start
```

##  Build
```
  yarn build
```

## Tests
```
  yarn test
```
